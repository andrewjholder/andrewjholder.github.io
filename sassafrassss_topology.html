06   SPECIAL CONFIGURATIONS & NOTES
📡 mDNS Repeater
Settings → Networks → mDNS → ON
Bridge: Main ↔ IOT
Bridge: Media ↔ IOT
Required for Sonos + Chromecast discovery across VLANs. Without this, Shared_Media firewall rule alone will not work.
⚠ IGMP Snooping
Settings → Networks → Multicast Settings
IGMP Snooping → OFF for IOT + Media VLANs
Multicast to Unicast → OFF on Sassafrassss_Home SSID (WiFi → Advanced)
UniFi aggressive IGMP drops multicast packets Sonos + Chromecast rely on.
⚠ Apple Private MAC
Must disable on every Apple device after first connect to each SSID.
iPhone/iPad: Settings → WiFi → ⓘ → Private Wi-Fi Address → OFF
Mac: System Settings → WiFi → Details → uncheck Private Address
Apple Watch: Settings → WiFi → network → Private Address → OFF
🌐 OpenDNS FamilyShield
VLAN 40 (Kids/Restricted) only
DNS: 208.67.222.123
Set in: Networks → VLAN 40 → DNS
Free, no account needed. DNS-level content filtering. Cannot be bypassed by kids.
🔄 IOT Device Onboarding
1. Connect to Sassafrassss_Home → lands on VLAN 50
2. Complete device setup via app (fully functional)
3. UniFi → Clients → device → Network Override → VLAN 30 or 60
4. Device reconnects to correct VLAN automatically
